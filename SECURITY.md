# Security Policy

## Reporting a Vulnerability
Contact: aotrust.near on NEAR AI Discord or legal@aotrust.link

## Safety Guarantee
A&O Trust Layer skills and MCP server will NEVER:
- Request your private key, seed phrase, or wallet password
- Execute shell commands or install software
- Access files outside the notarization workflow

## Verification
All PDR receipts are verifiable at:
https://api-staging.aotrust.link/v1/pdr/verify/{pdr_hash}
